# Demos

## 2024-25-03
Controls :
- Move : left stick
- Look : right stick
- Jump : south button
- Slide : left trigger when moving
- Slide Jump : south button when sliding
- Boost : right trigger when moving (needs energy)
- Jetpack : right trigger when in air (needs energy)
